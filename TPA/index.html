TPA · TI Project Assistant

一行命令，从 SysConfig 图形化配置出发，自动生成完整的 TI MSPM0 嵌入式项目。告别 CCS 手动配置。

仓库地址：GitHub | PyPI

功能特性

特性 说明
🔌 芯片支持 SDK 中全部 58 款 MSPM0 芯片，运行时动态发现
🛠 调试器 CMSIS-DAP / XDS110 / JLink / none 四种模式
📁 项目结构 分层 inc//src/，config/ 隔离自动生成代码
⚙️ CMake file(GLOB_RECURSE) 递归链接，arm-none-eabi-gcc 工具链
🖥 VSCode 集成 全套 .vscode/ 配置（launch / tasks / settings / c_cpp_properties）
🔄 regenerate 修改引脚后一键重生成，手写代码不受影响
🐙 Git 检测到 Git 时自动 init + .gitignore
🔘 状态栏按钮 配合 Task Buttons 插件，SysConfig / Build / Flash / Clean 一键操作
🌍 跨平台 Windows / Linux，PATH 优先自动发现工具链
🔍 自检 mspm0-init --check 一键诊断环境状态

环境依赖速查

Windows

依赖 版本 安装方式
Python ≥ 3.8 winget install Python.Python.3.12
ARM GCC 13.x+ ARM 官网 → bin\ 加入 PATH
CMake ≥ 3.13 winget install Kitware.CMake
Ninja 任意 winget install Ninja-build.Ninja
TI MSPM0 SDK 2.x 下载 → 解压到 C:\ti\
TI SysConfig 1.x 下载 → 解压到 C:\ti\
OpenOCD 1.3.x PATH 发现 / TI VSCode 插件（CMSIS-DAP/XDS110 必装，JLink 可选）
arm-none-eabi-gdb 14.x PATH 发现 / TI VSCode 插件
JLink 7.x+ SEGGER 官网 → 自动加入 PATH（JLink 用户必装）
Git 任意 winget install --id Git.Git -e --source winget

Linux

依赖 版本 安装方式
Python ≥ 3.8 系统自带 / sudo apt install python3
ARM GCC 13.x+ sudo apt install gcc-arm-none-eabi
CMake ≥ 3.13 sudo apt install cmake
Ninja 任意 sudo apt install ninja-build
TI MSPM0 SDK 2.x 下载 → 解压到 ~/ti/
TI SysConfig 1.x 下载 → 解压到 ~/ti/
OpenOCD 1.3.x PATH 发现 / TI VSCode 插件（CMSIS-DAP/XDS110 必装，JLink 可选）
arm-none-eabi-gdb 14.x PATH 发现 / apt install gdb-multiarch
JLink 7.x+ SEGGER 官网 → .deb 或解压到 /opt/

环境变量

全部可选。不设置时 TPA 从 PATH 自动发现。

变量 Windows 默认值 Linux 默认值 说明
MSPM0_SDK C:\ti\mspm0_sdk_* ~/ti/mspm0_sdk_* SDK 根目录
SYSCONFIG_DIR C:\ti\sysconfig_* ~/ti/sysconfig_* SysConfig 目录
TI_ROOT C:\ti ~/ti TI 工具根目录
OPENOCD_DIR PATH → TI 插件缓存 PATH → ~/.config/... OpenOCD 目录
GDB_DIR PATH → TI 插件缓存 PATH → ~/.config/... GDB 目录
JLINK_DIR PATH → C:\Program Files\SEGGER\JLink PATH → /opt/SEGGER/JLink* JLink 目录

支持的芯片

SDK 中的全部 58 款 MSPM0 芯片开箱即用。 TPA 通过运行时解析 SDK 自动发现芯片参数，无需手动添加型号。

系列 代表型号
MSPM0Gx5xx G3507, G3519, G3107, G1507, G1519 …
MSPM0Gx1xx G1105, G1106 …
MSPM0Lx3xx L1306, L1345, L2228 …
MSPM0Lx2xx L1227, L2227, L1228 …
MSPM0Cx1xx C1105, C1106 …
MSPM0Hx2xx H3215 …
