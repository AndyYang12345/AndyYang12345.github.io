项目概述

面向 RoboMaster 竞赛场景的半自主移动操作平台，实现底盘运动、机械臂抓取、能量机关收集与提交等复杂任务。

  • 轮式差分驱动底盘 + 5-DOF 机械臂 + 两指对称夹爪 + 可旋转回收杆
  • ROS 2 Jazzy 分布式架构，真机部署 Raspberry Pi 4B + STM32
  • 手柄遥控 + 视觉辅助双模融合，半自主任务执行
  • 独立视觉模块，基于 MaixCam2 嵌入式平台，仿真预研后再上实机

🎮 输入抽象层

  • JoystickIntent：死区检测 + NORMAL/CENTERED/EDGE 状态
  • ButtonIntent：按下/释放边缘触发
  • ComboIntent：多按钮组合（双扳机折叠手臂）
  • TriggerIntent：LT/RT 0~1 线性值
  • 速率过滤：指数平滑抑制抖动

仿真层详解

🎨 靶子建模 (TargetSim)

1中心 + 5外围五角星布局，12种预定义 BGR 颜色，暴露 Ground Truth 便于算法评估

🎬 流式帧生成器
